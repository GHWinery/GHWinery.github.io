Order Completion
Complete the form by marking the items you need ordered:
• Enter the Current Stock of the item
• Check the Order checkbox
• For wine orders, enter Cases Ordered
• Enter any additional details

Order Confirmation
Send a message in the Square Production Chat confirming your order is finished
Store Lead / Manager
Note: The deadline to order for the week is 3 PM on Tuesday. Changes made to the spreadsheet after 3 PM might not be fulfilled until the following week.
Mark Items Received
After orders are fulfilled and delivered, open the Winery Supply Order Form and mark items as received by unchecking the Order checkbox

If any items were not delivered, leave the Order checkbox marked

4. Fulfilling Wine Orders for Store Locations

Procedure / Form
Process Step / Activity
Responsible Function
Order Receipt
Receive completed Order Form from Supply Chain Manager on Tuesday afternoon
Production Team
Order Preparation
Prepare delivery according to order form:
• Print order form
• Gather wine and supplies for the store based on the order form
• Place items near garage door for loading
• Attach order form to the delivery
• Repeat for every store's order
Production Team
Delivery Preparation
On Wednesday, load orders into van based on route
Production Team

Deliver supplies to store
